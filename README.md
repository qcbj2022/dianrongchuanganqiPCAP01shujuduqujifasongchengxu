# 电容传感器PCAP01数据读取及发送程序

## 项目描述

本项目提供了一个基于STM32F103主控芯片的电容传感器PCAP01数据读取及发送程序。通过SPI总线与PCAP01传感器进行通信，实现对传感器寄存器的读写操作，从而设置传感器的刷新频率和精度等参数。读取到的电容数据将通过485总线传输出去。

## 功能特点

- **SPI通信**：使用STM32F103的SPI接口与PCAP01电容传感器进行通信，实现对传感器寄存器的读写操作。
- **参数配置**：支持设置传感器的刷新频率和精度，以满足不同应用场景的需求。
- **数据传输**：通过485总线将读取到的电容数据传输出去，便于后续的数据处理和分析。

## 使用说明

1. **硬件连接**：
   - 将STM32F103与PCAP01传感器通过SPI总线连接。
      - 将STM32F103与485总线模块连接，确保数据传输的稳定性。

      2. **软件配置**：
         - 根据实际需求，配置传感器的刷新频率和精度参数。
            - 编译并烧录程序到STM32F103开发板。

            3. **数据读取与传输**：
               - 程序运行后，STM32F103将通过SPI总线读取PCAP01传感器的电容数据。
                  - 读取到的数据将通过485总线传输出去，供外部设备接收和处理。

                  ## 注意事项

                  - 确保硬件连接正确，避免因连接错误导致的数据读取失败。
                  - 根据实际应用场景调整传感器的参数设置，以获得最佳的测量效果。
                  - 在数据传输过程中，注意485总线的通信距离和环境干扰，确保数据传输的可靠性。
